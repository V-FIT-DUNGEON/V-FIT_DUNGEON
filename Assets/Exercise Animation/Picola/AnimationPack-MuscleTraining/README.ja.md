## Description
筋トレのモーションパックです
このパッケージには、「腹筋」「レッグレイズ」「懸垂」「腕立て伏せ」「スクワット」の5種類のモーションと、それぞれに筋肉を酷使してプルプルと震えるバリエーションの計10種類が含まれています。

このアセットには、下記のアニメーションが含まれます。
標準的なHumanoidリギングモデル用

- anm_chinup01_01

- anm_pushup01_00
- anm_pushup01_01
